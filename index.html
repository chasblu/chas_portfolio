Top Anime

Link to site

Link to Github

Top Anime uses the jikon.moe API to generate anime data. On the home page the top 3 anime ranked by My Anime List(a popular anime site) are populated with links to the corresponding My Anime List page. There is a Category page that generates a list of 100 anime based on the category that you select. The category results are displayed by popularity. There is a search page that allows you to search for your favorite anime and displays a list of 50 results that match by title. The search results are displayed by closest matching title.
